# Sample Template Documents

This folder contains sample template documents that users can use with the Template Doc feature.

## Available Templates

### 1. Research Summary Template
A simple template for basic research summaries.

**Sections:**
- Title
- Authors
- Abstract
- Journal Information
- Publication Date
- PMID

### 2. Detailed Research Report Template
A comprehensive template for detailed research documentation.

**Sections:**
- Study Information
  - Title
  - Authors
  - Publication Details
- Abstract
  - Importance
  - Objective
  - Design
  - Results
  - Conclusions
- Reference Details
  - Journal
  - Publication Date
  - PMID
  - URL

### 3. Citation Template
A formatted citation template.

**Sections:**
- Citation
  - Authors
  - Title
  - Journal
  - Publication Date
  - PMID

## Creating Your Own Template

1. Create a new Word document (.docx)
2. Use Word's built-in heading styles:
   - **Heading 1** for main sections
   - **Heading 2** for subsections
   - **Heading 3** for sub-subsections
3. Use clear, standard academic terms
4. Save as .docx format
5. Upload through the Template Doc interface

## Supported Heading Names

The system automatically recognizes these heading names:

- **Title**: title, article title, study title, name
- **Authors**: authors, author, researchers, investigators
- **Journal**: journal, publication, source
- **Date**: publication date, date, year, published
- **PMID**: pmid, pubmed id, id, identifier
- **Abstract**: abstract, summary, overview
- **Objective**: objective, aims, purpose, goal
- **Methods**: design, study design, methodology, methods
- **Results**: results, findings, outcomes
- **Conclusions**: conclusions, implications, summary
- **Background**: introduction, background, context

## Tips

- Keep heading names simple and clear
- Use standard academic terminology
- Test with preview before generating
- Start simple and add complexity gradually
